Images should ALWAYS include an alt attribute (even if it's going to be null!!!) in order to ensure that someone using a screen reader will have something suitable displayed, but that's not all. They're also useful in search-engine optimization, as well as display things when you have broken links.

"alt" means 'alternative text'. Don't use "style" anymore. 'accesskey' allows a shortcut to an element. "class" applies special properties to groups of elements. "tabindex" wth is that???

There are LOTS of "special entities" that control how certain characters are displayed in a browser...this class is obviously not the place where all of them can be listed.
